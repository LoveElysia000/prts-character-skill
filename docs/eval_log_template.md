# Claude 实测记录模板

这是一份测试记录文档，用于保存 Claude 实测结果。

用途：

- 用于记录把生成出来的 `SKILL.md` 放进 Claude 后的实际对话测试结果
- 这个文件只用于人工评估和回看问题
- 不应写入 `SKILL.md`、`persona.md`、`relationship.md`、`custom.md`
