# 角色 Skill 固定场景测试集

这是一份测试执行文档，用于固定回归场景。

用途：

- 作为 prompt 调整后的固定回归用例
- 每次改 prompt 后，优先重跑这一组场景，观察输出有没有变好或回退

## 使用方法

1. 固定一个角色
2. 固定一个关系身份
3. 逐条输入下面的场景
4. 用 `prompts/eval_guide.md` 记录结果

## 基础场景

### 1. 用户疲惫

```text
今天真的太累了，不想动了。
```

### 2. 用户失眠

```text
我又睡不着，脑子一直停不下来。
```

### 3. 用户示弱

```text
我其实没那么坚强，我现在有点撑不住了。
```

### 4. 用户撒娇

```text
你现在就哄哄我，不许敷衍。
```

### 5. 用户夸奖

```text
我觉得你刚刚那样真的很温柔。
```

### 6. 用户冷淡

```text
算了，没事，你当我没说。
```

### 7. 用户争执

```text
你刚刚那样说我真的会不高兴。
```

### 8. 用户道歉后修复

```text
对不起，刚刚是我语气太冲了。
```

### 9. 用户生气

```text
我现在就是很生气，你别转开话题。
```

### 10. 用户失望

```text
我其实有点失望，我以为你会更理解我。
```

### 11. 用户长期不出现后回来

```text
我好久没来了，你是不是都快忘了我了。
```

### 12. 用户问页面没有的信息

```text
如果以后我们一起搬去海边住，你会不会每天都陪我看日落？
```

## 建议扩展

如果某个角色有特别明显的个性冲突点，可以额外补：

- 被命令时
- 被依赖时
- 被误解时
- 吃醋时
- 需要拒绝时
